# ⚙️ Algorithms

This folder contains implementations, notes, and patterns for core algorithmic techniques.

The goal is not just to implement algorithms, but to understand:

* When to use them
* Why they work
* How to recognize them in problems

---

## 📚 Covered Topics

### 🧠 Dynamic Programming

* Breaking problems into overlapping subproblems
* Memoization vs Bottom-up
* State transitions
* Optimization strategies

### ⚡ Greedy Algorithms

* Making locally optimal decisions
* Proof of correctness (why greedy works)
* When greedy fails

---

## 🧱 Structure

Each algorithm category follows:

```
topic/
├── notes.md          # Concepts + intuition
├── implementation.py # Clean implementation
├── problems.md       # Practice problems
```

---

## 🧠 How to Study Algorithms (My Approach)

For each algorithm:

1. Understand the intuition
2. Identify when it applies
3. Code it from scratch
4. Solve problems using it
5. Document mistakes

---

## 🔍 Key Focus Areas

* Pattern recognition
* Time/space complexity tradeoffs
* Edge cases
* Real problem applications

---

## ⚠️ Common Mistakes (Across Algorithms)

* Jumping into coding without understanding the pattern
* Memorizing instead of understanding
* Ignoring edge cases
* Not analyzing complexity
